1、filestream API

  • 读文件：fs.readFile（异步）
  • 写文件：fs.writeFile（异步）
  • 追加内容：fs.appendFile（异步）

2、集成例子

通过访问不同的querystring来操作不同的API：

  • localhost:2014?read => 读文件
  • localhost:2014?write => 写文件
  • localhost:2014?append => 追加内容
